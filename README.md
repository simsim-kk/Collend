# ◇ Collend

**粘贴即收藏，AI 帮你整理**

本地收藏 · 隐私在端 · 零服务器

🌐 [collend.netlify.app](https://collend.netlify.app)

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 📝 自动摘要 | 长文与网页一键提炼要点 |
| 🏷️ 智能标签 | 自动提取关键词，点按即可筛选 |
| 🔍 全文搜索 | 标题、摘要、标签与原文一搜即达 |
| 💾 导出备份 | JSON 导出与导入，数据随身携带 |
| 🗒️ 个人备注 | 每条收藏独立笔记，随写随存 |
| 📌 收藏置顶 | 重要条目始终排在列表最前 |

---

## 使用方式

1. 打开 [collend.netlify.app](https://collend.netlify.app)
2. 粘贴一段文字，或一个 `https://` 链接
3. 点击「解析并保存」，AI 自动生成摘要与标签
4. 搜索、筛选、备注，随用随取

---

## 数据隐私

所有数据**仅保存在你的浏览器本地存储**中，不会上传到任何服务器。
